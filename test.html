Text-to-Speech Extension Test Page

How to Test the Extension:

  1. Make sure the Text-to-Speech Reader extension is installed
  2. Click the extension icon in your Chrome toolbar
  3. Configure your voice settings
  4. Click "Start Reading" to test the functionality
  5. Try the floating controls by clicking "Show Floating Controls"

Sample Content for Testing

This is a test page to verify that the Text-to-Speech Reader Chrome extension is working correctly. The extension should be able to read this content aloud when activated.

The extension uses the Web Speech API to convert text to speech. It intelligently extracts the main content from web pages while excluding navigation elements, advertisements, and other non-content areas.

Features to Test:

  • Voice selection from available system voices
  • Speed adjustment (0.5x to 2x)
  • Pitch adjustment (0.5 to 2)
  • Volume control (0 to 1)
  • Floating controls that can be moved around the screen
  • Progress tracking through the content
  • Navigation between text chunks

The extension should automatically detect this main content area and read it aloud. The floating controls provide additional functionality like play/pause, stop, and navigation between different sections of the text.

Technical Details:

This extension is built using Manifest V3 and includes:

  • A popup interface for configuration
  • A content script that runs on web pages
  • A background service worker for extension management
  • Floating controls that can be dragged around the page
  • Settings persistence using Chrome's storage API

If everything is working correctly, you should hear this content being read aloud when you activate the extension. The voice should be clear and the reading should be smooth and natural.
